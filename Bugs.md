# Отчет о багах

## Критические баги (High)

### 1. Некорректная работа фильтра "Формат работы"
Применяя фильтр "Можно удаленно" отображаются ватансии с разными форматами работы.

### 2. Некорректная работа фильтра "Направление"
При выборе направления "Data Science" отображаются вакансии из других направлений.

### 3. Некорректная работа фильтра "Город"
При выборе города "Санкт-Петербугр" отображаются вакансии из других городов.

### 4. Некорректная работа фильтра "Руководящая роль"
При веборе пункта "Руководящая роль" отображаются сторонние вакансии.

### 5. Ошибочное отображение окна отсутствия вакансий
Строка "Ничего не нашлось" отображается при наличии найденных вакансий.

### 6. Некорректная работа счетчика вакансий у группировки по Направлениям
Числа в счетчике в направенни "Клиентский сервис" не равно числу вакансий в группе (2 вместо 1).

### 7. Перепутанны команды в вакансиях
У вакансии "Аналитик данных в бизнес-команду Авито Авто" указана команда "Недвижимость", а у вакансии "Руководитель аналитики в транзакционное направление жилой недвижимости" указана команда "Авто".

## Существенные баги (Medium)

### 8. Отсутствие изображения на кнопке очиски поля ввода вакансии 
Рядом с полем поиска по названию вакансии отсутствует иконка крестик.

### 9. Лишний тег в строке тэгов
В блоке выбранных фильтров отображается лишний тег "Поддержка пользователей".

### 10. Неверное уведомление в поле выбора коннанды 
Отобразается уведомление "Вакансий пока нет" вместо "Нет команды для этих вакансий"

### 11. Отсутствие иконки стрелки на кнопке "Про направление"
На кнопке "Про направление" у направления "Аналитика данных" отсутствует иконка стрелки.

### 12. Отсутствие города у вакансии
У вакансии "Аналитик данных в бизнес-команду Авито Авто" отсутствует город.

## Косметические баги (Low)

### 13. Отсутствие выравнивания вакансий
Карточки вакансий не выравнены по вертикали относительно друг друга.

### 14. Неверный адрес Офиса в Москве
Указан адрес "ул. Лесная, 77" вместо корректного "ул. Лесная, 7".

## Описание приоритетов багов

1. Критические баги (high) - баги, которые не позволяют пользователю использовать функциональность поиски вакансий на сайте (Ошибки при работе фильтров).
2. Существенные баги (medium) - неудобства в использовании сайта.
3. Косметические баги (low) - визуальные дефекты, которые не влияют на функциональность сайта.
